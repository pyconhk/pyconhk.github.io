Workshop: Massive Open Online Course Architecture using Python + Web Platform

Speaker: Rizky Ariestiyansyah (ariestiyansyah)

Language: English (with English Slides)

Beginner

About Speaker

Mozilla Tech Speaker and Chief Technology Officer of Indonesia

About the Topic

Education is very important for everyone, but many obstacles where media for education is still lacking and the adoption of the technology used is still small. Through this session, participants will learn the concept of MOOC, making MOOC and combine it with web technology. With the MOOC everyone can freely access education. Participants will also learn how the production of courses/materials that will be used in MOOC. The main focus is [articipants can use Python, Django and WebAPI to build MOOC Platform.
