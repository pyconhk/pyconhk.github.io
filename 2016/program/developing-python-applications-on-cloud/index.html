Keynote talk: the Pandas project and its future

Speaker: Xiulei ZHU

Language: Mandarin

Beginner

About Speaker

Xiulei ZHU, IBM Bluemix Engineer, Bluemix System/Automation Testing Lead, IBM Greater China

Xiulei is passionate in open cloud (Cloud Foundry/Docker) technology. He started working on IBM Cloud platform - Bluemix from 2013. Prior to that, his focus areas were middleware (IBM Websphere Application Server). He is also IBM developerWorks Professional Authors. You may get to know more about him and his articles/projects from developerWorks website and Github (acostry).

About the Topic

Description: in this session, Xiulei is going to demonstrate how to create and run Python web applications and develop business intelligence/analytic applications with Apache Spark and Python on IBM Bluemix cloud platform.

Tags: Cloud, IBM Bluemix, Python
